# Snowball Models

Snowball 앱과 서버 간 공유하는 UI 모델 라이브러리입니다.

## Features

- Kotlin Serialization 지원
- KMP (Kotlin Multiplatform) 호환
- 순수 데이터 모델 (비즈니스 로직 없음)

## Models

- `InvestmentStatusUiModel`: 투자 상태 UI 모델
- `HomeUiModel`: 홈 화면 UI 모델
- `StockDetailUiModel`: 주식 상세 화면 UI 모델
- `TradeHistoryUiModel`: 거래 내역 UI 모델

## Dependencies

- Kotlin 2.2.20
- Kotlin Serialization
- Ktor Serialization Kotlinx JSON

## Version

0.1.2
